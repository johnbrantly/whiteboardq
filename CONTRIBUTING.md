# Contributing

This project is not currently accepting pull requests or external contributions.

You are welcome to:

- **Report bugs** via [GitHub Issues](https://github.com/johnbrantly/whiteboardq/issues)
- **Fork** and build upon this project under the terms of the [GPL v3.0](LICENSE)

If you've found a security vulnerability, please see [SECURITY.md](SECURITY.md) for reporting instructions.
